# RG-09 Module Definitions

This directory contains production JSON template-definition files for RG-09:

- `dmb_basel3.json` (`DMB_BASEL3`) - 15 sheets
- `ndic_returns.json` (`NDIC_RETURNS`) - 11 sheets
- `psp_fintech.json` (`PSP_FINTECH`) - 14 sheets
- `pmb_cbn.json` (`PMB_CBN`) - 12 sheets

Total across RG-09 definitions:

- Templates: 52
- Fields: 1717
- Formulas: 475
- Cross-sheet rules: 80
- Inter-module data flows: 28

Traceability notes for regulatory source templates in `/Templates`:

- DMB legacy workbook sheets `AQR`, `FXP`, `LEX`, `STR` are explicitly folded into RG-09 templates (`DMB_IFR`/`DMB_NPL`, `DMB_MKR`, `DMB_LND`, `DMB_CAP`) with `sourceSheetsCovered` metadata and dedicated fields/formulas.
- NDIC legacy workbook sheets `SPC`, `EXM`, `CLM`, `RPT` are explicitly folded into RG-09 templates (`NDIC_EWS`, `NDIC_GOV`, `NDIC_PAY`, `NDIC_DIC`) with `sourceSheetsCovered` metadata and dedicated fields/formulas.

The files are generated by:

`scripts/generate_rg09_module_definitions.py`

To regenerate:

```bash
cd "RegOS™"
python3 scripts/generate_rg09_module_definitions.py
```
